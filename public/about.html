ABOUT

This is a demo application written using the MEAN stack specifically built to demonstrate:

  • Implementation of Bootstrap dropdown via Angular directives
  • Angular based todo app
  • Express, Mongo, and Node based api

I prioritized speed of implementation, while exploring the MEAN stack. It took 5 hours, over two sessions, to build and finish up. About half the time was spent learning about the stack and the other half in coding.

Much of the code based on tutorial and example code from:

  • Scotch.io Tutorials
  • Boostrap and Angular Bootstrap UI Documentation
  • Stack Overflow solutions
